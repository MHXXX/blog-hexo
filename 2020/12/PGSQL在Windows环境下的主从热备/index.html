2. 开始备份

  • 连接到主库, 执行命令 select pg_start_backup('Replitionwork');开启备份
  • 关闭备库, 清空备库 data 目录, 拷贝主库 data 目录下的所有文件到备库 data 目录下, 删除 postmaster.pid 文件
  • 在主库执行命令 select pg_stop_backup(),current_timestamp; 关闭备份
