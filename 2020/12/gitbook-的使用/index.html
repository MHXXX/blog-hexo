2020-12-24
gitbook

安装

  • 需要安装 nodejs , 使用 node -v 来检测
  • 执行命令 npm install gitbook-cli -g 进行安装
  • 执行命令 gitbook init {dir} 下载 gitbook 并在指定目录下初始化
  • 执行命令 gitbook server 以服务的形式启动 , 浏览器访问 127.0.0.1:4000
  • 执行命令 gitbook build 完成 html 的建立
