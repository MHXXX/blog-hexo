linux 上部署

  • 目前新版本的 elasticsearch 自带 java 运行环境, 所以不需要配置 JAVA_HOME
  • 先从 官网 下载最新的安装包并解压
  • 创建一个非 root 用户 (例如 es) 并清除密码, 并将解压后的目录赋给 elastic
  • 调整 vm.max_map_count 的值为262144
  • 进入 elasticsearch 的 bin 目录运行./elasticsearch 直到能成功运行
  • 编写开机自启脚本, 需要使用 su -c "elasticsearch -d" {user} 使其后台运行
